Eventos JavaScript

Evento Ejemplo de aplicación Demostración
mouseenter Mostrar una imagen ampliada cuando el usuario coloca el cursor sobre ella.
mouseleave Ocultar un elemento cuando el cursor sale de su área.
focus Cambiar el estilo de un campo de texto cuando recibe el enfoque.
mouseout Se activa cuando el puntero del mouse se aleja de un elemento.
Aleja el cursor
mouseover Se activa cuando el puntero del mouse se mueve sobre un elemento.
Pasa el cursor aquí
keydown Se activa cuando se presiona una tecla en el teclado. Presiona una tecla
resize Ajustar el diseño de una página en respuesta al cambio de tamaño dela ventana del navegador.
contextmenu Personalizar el menú contextual que aparece al hacer clic derecho enun elemento.
Haz clic derecho aquí para mostrar el menú contextual personalizado.
touchstart Responder al inicio de un toque en una pantalla táctil.
Toca aquí para iniciar el evento táctil.
scroll Realizar una acción cuando el usuario desplaza el contenido de una página.
